• 方法 1: 利用技巧

      var a = 5;
  var b = 6;
  a = a + b;
  b = a - b;
  a = a - b;

  console.log(a); // 6
  console.log(b); // 5

  • 方法 2: 利用位运算

      var a = 5; // 二进制101
  var b = 6; // 二进制110

      110

  a = a^b;
=>
  a = 011
      101
      ━━━
      110

  console.log(a); // 6
  console.log(b); // 5
