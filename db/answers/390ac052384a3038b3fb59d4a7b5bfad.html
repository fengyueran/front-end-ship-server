• 选择排序算法

    基本思想:

    • 在无序数列中寻找最小或最大的数，将其存放到排序数列的起始位置。
    • 继续在未排序的数列中查找最大(或最小)的元素，存放到已排序数列的末尾，也就是查找的起始位置
    • 如此往复直到排序完毕
    var selectionSort = arr => {
  let tmp;
  const len = arr.length;
  for (let i = 0; i < len - 1; i++) {
    let minIndex = i;
    for (let j = i + 1; j < len; j++) {
      if (arr[j] < arr[minIndex]) {
        minIndex = j;
      }
    }
    tmp = arr[i];
    arr[i] = arr[minIndex];
    arr[minIndex] = tmp;
  }
  return arr;
};
