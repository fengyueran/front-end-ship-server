水平居中方法

    1. flex
    • flex 2009
       display: box;
   box-pack: center;
    • flex 2012 A: display: flex; justify-content: center; 子元素 B 水平居中 B: display: flex; justify-content: center; 子元素 C 水平居中
    1. text-align 只对内联元素有效， 块级元素想要通过 text-align 来居中可以将其 display 属性设置为 inline。 A: text-align: center; 行内元素 B 水平居中
    1. margin-auto 对块级元素有效，需要设置宽度，行内元素若要通过 margin 居中可以设置其 display 属性为 block。 B(子元素): width: 200px; margin: 0 auto; 块级元素 B 水平居中，
    1. position 子元素 B 相对于父元素 A 的 left 为父元素宽度减去子元素宽度的一半， 因此必须知道父子元素的宽度。 A:width: 500px; position: relative; B: position: absolute; left: 180px; width: 140px;
    1. position + margin-left 不需要知道父元素的宽度，通过 margin-left 设置为自身的一半宽度的负值达到向左偏移自身的一半宽度， 与 4)中的实现方法本质是一样的，但需要知道子元素 B 的宽度。 A: position: relative; B: position: absolute; left: 50%; margin-left: -70px; width: 140px;
    1. position + transform 不需要知道父子元素的宽度，先定位到父元素的水平中心位置再通过 transform 向左偏移自身的一半宽度， 与 4)中的实现方法本质是一样的。transform 只对块级元素有效。 A: position: relative; B: position: absolute; left: 50%; transform: translateX(-50%);
    1. margin-auto + fit-content fit-content 是 CSS3 中给 width 新加的一个值，表示将元素宽度收缩为内容宽度。 如下，li 为浮动元素，要让导航栏 ul 居中，尝试 3)中的方法，设置 ul 的 margin 为margin: 0 auto，然而并没有什么用，why？因为 ul 并没有设置宽度，但是这种情况下我们很难去计算 ul 的宽度，fit-content 闪亮登场，设置 width 为 fit-content 即给 width 设置了一个宽度，从而达到了效果。
      // li { float: left }
<div>
 <ul>
    <Li><a href="/">Meat</a></Li>
    <Li><a href="/">Fruits</a></Li>
    <Li><a href="/">Vegetables</a></Li>
 </ul>
</div>
      A(div): width: 500px; B(ul): width: fit-content; margin: 0 auto;

垂直居中方法

    1. flex 2012 A: display: flex; align-items: center; 子元素 B 垂直居中 B: display: flex; align-items: center; 子元素 C 垂直居中
    1. position 子元素 B 相对于父元素 A 的高 top 为父元素高度减去子元素高度的一半，因此必须知道父子元素的高度 A: position: relative; height:150px; B: position: absolute; top: 50; height: 50px;
    1. position + margin-top 子元素 B 相对于父元素 A 的 top 为父元素高度的一半，margin-top 为子元素高度一半的负值，因此必须知道子元素的高度。 A: position: relative; height:150px; B: position: absolute; top: 50%; margin-top: -25px; height: 50px;
    1. position + transform 子元素 B 相对于父元素 A 的 top 为父元素高度的一半，通过 translateY(-50%)使子元素向上偏移自己一半的高度，本质上与 3）、4）中的方法一样。 不需要知道父子元素的高度。 A: position: relative; B: position: absolute; top: 50%; transform: translateY(-50%);
    1. position + margin-auto 如下，父元素 A 为相对定位，子元素 B 为绝对定位，设置 B 的 top、bottom 都为 0，并设置其 margin 为 auto 0 即可使子元素 B(高度固定)垂直居中。 A: position: relative; B: position: absolute; top: 0; bottom: 0; height: 50px; margin: auto 0;

    1. vertical-align vertical-align 用于对齐行内元素(inline、inline-block、inline-table)，当父元素为 td、th 且高度固定时可以设置 vertical-align 为 middle 来垂直对齐子元素。

      结构: 
      <div>
  <table>
   <td>
     <th>
       <B />
     </th>
   </td>
  </table>
<div>

      A(th): height: 70px; vertical-align: middle;

    1. inline-block + vertical-align 父元素 A 添加伪元素，伪元素与父元素同高且为行内块级元素、居中对齐，子元素 B 设置为行内块级元素，居中对齐。 A(div): &:after { content: ""; height: 100%; vertical-align: middle; display: inline-block; } B(div) : vertical-align: middle; display: inline-block;
    1. line-height 对于单行文本，我们可以通过设置其 line-height 为父元素高度，使其垂直居中。 A(div): height: 100px; B(span): line-height: 100px;
