html、css、js

  • W3C W3C 组织是制定网络标准的一个非赢利组织，像 HTML、XHTML、CSS、XML 的标准就是由 W3C 来定制，该网站可以查询到相关的标准。

  • caniuse 这个网站可以查询各个浏览器对 CSS、JS 的支持情况。

  • HTML Dog 这个网站汇集了 html、css 以及 js 大量的教程资源。

  • single div 如其名，这个网站有很多作者用单个 div 制作的精美的图形，非常有意思。

  • You-need-to-know-css 这个网站有很多有意思的 css 用法。

  • medium medium 是一个内容平台，里面的文章质量较高，推荐阅读。

前端期刊

  • Html5 Weekly 老牌前端周刊，以介绍 HTML/CSS 技术的文章为主，原来叫做 html5 weekly，现在改为 FrontEnd Focus，rss 订阅地址为https://frontendfoc.us/rss。

  • javascriptweekly javascript 相关内容，质量很高，rss 订阅地址为https://javascriptweekly.com/rss。

  • css-weekly 老牌 CSS 期刊，主要介绍 CSS 相关内容。也介绍一些不错的工具和开源项目，比较适合于前端新人。rss 订阅地址为https://feeds.feedburner.com/CSS-Weekly。

  • web-design-weekly 前端开发内容居多，包含设计、技术、技巧、工具等内容，rss 订阅地址为https://web-design-weekly.com/write-web-design-weekly/。

  • webtoolsweekly 主要推送一些工具相关的内容，rss 订阅地址为http://feeds.feedburner.com/WebToolsWeekly。

  • 掘金 国内还不错的技术分享交流平台，有不少高质量的文章，rss 订阅地址为https://juejin.im/rss。

  • 众成翻译 中国本土的技术翻译社区，翻译了不少高质量的文章

图标

  • icons8 这个网站提供免费的平面设计图标为主，总数将近四万，支持分类搜索，并可以选中合适的图标尺寸、格式进行下载。

  • iconfont 这个网站是阿里巴巴矢量图标库

其他

  • js 调查报告 State of JavaScript 是为了了解全球 js 发展趋势而创建的，包括前后端、桌面、移动端等的内容。每年都会出一个报告，2018 年的报告网址为: https://2018.stateofjs.com，2019 年的调查报告应该就是https://2019.stateofjs.com了。
