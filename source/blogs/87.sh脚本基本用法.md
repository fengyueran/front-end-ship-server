- if-else

  ```sh
  //build目录是否存在
    if [ -d build ]; then
      echo true

    fi
  ```

- if-elseif

  ```sh
  project="ops-next"
  if [ $1 = "order" ]; then
    project="dv-order"
  elif [ $1 = "dvfcloud" ]; then
    project="ops-next-gateway"
  fi

  echo $project
  ```
